# CLAW - P.A.R.A. Organized Version

Este repositorio ha sido organizado siguiendo el estándar **P.A.R.A.** y la nomenclatura **ISO-SAGE**.

## Estructura P.A.R.A.

- **00_SOPORTE/**: Configuraciones, requisitos, entorno y logs.
- **01_SRC/**: Lógica principal del asistente (ClawSpring v3.05.5).
- **02_TESTS/**: Pruebas unitarias y de integración.
- **03_DOCS/**: Documentación, manuales y archivos históricos.
- **04_ASSETS/**: Recursos estáticos, imágenes y demos.

## Ejecución

Para ejecutar el asistente, utiliza el lanzador en la raíz:

```bash
python3 run_claw.py
```

## Nomenclatura ISO-SAGE

Los archivos siguen el formato: `CLAW_YYYY_MM_DD_DESCRIPCION_VXX.py`

## Reglas E-SYSTEM

Consulta el archivo `.clinerules` para las reglas de programación y coordinación de IAs.
